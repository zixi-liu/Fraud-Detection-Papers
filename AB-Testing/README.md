## Multi-Armed Bandit

Traffic is dynamically re-allocated to winning variants.

**When to Use Multi-Arm Bandit?**
- Winning over learning
- Single goal metric
- Multiple variations



## Platform-wise
随机对照试验 RCT
- 在线上流量中取出一小部分用户（较低风险），完全随机地分给原策略A和新策略B（排除干扰），再结合一定的统计方法，得到两种策略对比效果的准确估计。
